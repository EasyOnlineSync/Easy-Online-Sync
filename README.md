# Easy Online Sync

O **Easy Online Sync (Eos)** é um programa desenvolvido para Windows 11 que permite baixar arquivos de forma prática e segura.  
**Eos** foi projetado para funcionar em conjunto com a **extensão oficial para o Google Chrome**, já disponível para instalação.

## Recursos principais
- Sincronização rápida entre o programa e o navegador.
- Integração direta com a extensão do Chrome.
- Instalação simples via **Eos Setup.exe**.
- Suporte completo para Windows 11.
- Compatibilidade parcial com Windows 10 (pode apresentar alguns bugs visuais).

## Extensão para Chrome
Para que o Eos funcione corretamente e baixe arquivos, você precisa instalar a extensão:  
[Extensão Eos Analysis Module no Chrome Web Store](https://chromewebstore.google.com/detail/mckhcfhapcceifjnhbgehblphfbpfhcd?utm_source=item-share-cb)

## Instalação
1. Baixe o instalador **Eos Setup.exe** na seção de Releases.
2. Execute o instalador e siga as instruções. Durante a instalação, você provavelmente encontrará um erro de registro. Simplesmente ignore e continue com a instalação.
3. Após a instalação, abra o Chrome e instale a extensão.
4. Pronto! O Easy Online Sync estará integrado ao seu navegador.

## Sistemas suportados
- **Windows 11** → totalmente suportado.  
- **Windows 10** → funciona, mas pode apresentar alguns bugs visuais.

---

💡 Dica: sempre mantenha o Easy Online Sync e a extensão atualizados para garantir melhor desempenho e segurança.
